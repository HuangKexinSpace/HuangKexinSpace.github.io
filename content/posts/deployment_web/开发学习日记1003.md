# Cpp
## 2.3  复合数据类型
- 引用 
	- 引用就是**起别名**，引用的量必须先初始化。
	- 一旦引用，引用值就和初始值绑定在一起，对其所有的操作都是在与之绑定的对象上（不能和字面值或者表达式的结果bang hd）完成的。
	- 引用不是对象，没有引用的引用。
```
int ival = 6;
int &renameival = ival;
renameival = 999;
std::cout<<ival<<std::endl;
//输出会是999，所有的操作都是对ival完成的
```
